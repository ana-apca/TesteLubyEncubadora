# 🧪 Desafio Luby — Teste Técnico

Projeto desenvolvido como desafio técnico para a **Luby Encubadora**, demonstrando habilidades em desenvolvimento web full-stack e banco de dados.

## Sobre o Projeto

Aplicação web com operações CRUD completas e integração com banco de dados SQL Server, desenvolvida como parte de um processo seletivo técnico.

## Tecnologias

- **Frontend:** JavaScript · HTML · CSS
- **Backend:** ASP.NET / Node.js
- **Banco de Dados:** SQL Server
- **ORM/Queries:** SQL nativo

## Estrutura

```
├── DesafioLuby/       # Código-fonte da aplicação web
├── TesteLuby/         # Projeto de testes
├── TesteLuby.sql      # Scripts de criação do banco de dados
├── TesteLuby.txt      # Documentação do desafio
└── README.md
```

## Como Executar

1. Clone o repositório
2. Execute o script `TesteLuby.sql` no SQL Server para criar o banco
3. Abra a solution no Visual Studio
4. Configure a connection string
5. Execute o projeto
